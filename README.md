Syntax Overview (WIP)
===============================


Comments:
-------------------------------
    ; this is a comment
      
Numbers
-------------------------------
    1 + 2        ; = 3
    3 * 5        ; = 15
    6(8)         ; = 48
    (5 - 1) * 3  ; = 12
    5 - 1 * 3    ; = 2
    300 * 0.25   ; = 75
    
Booleans
-------------------------------
    true             ; = true
    false            ; = false
    3 < 4            ; = true
    8 == 9           ; = false
    1 > 2 and 3 < 4  ; = false
    5 > 6 or 1 < 2   ; = true

Lists
-------------------------------
    [ 1, 2, 3 ]             ; = [ 1, 2, 3 ]
    [ 5, 9, 4 ] . 0         ; = 5
    [ 5, 9, 4 ] . 1         ; = 9
    [ 5, 6, 2, 3, 5 ] .. 2  ; = [ 2, 3, 5 ]
    [ 1, 3, 2 ] + [ 7, 8 ]  ; = [ 1, 3, 2, 7, 8 ]
    [ 1, 2 ] == [ 1, 2 ]    ; = true
    [ 1, 2 ] == [ 2, 1 ]    ; = false
